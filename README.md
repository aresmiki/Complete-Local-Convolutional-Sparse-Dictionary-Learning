This is a fast dictionary learning algorithm library that enables rapid self-learning for local support of repetitive events or signals, significantly reducing memory consumption and accelerating learning speed. Using this method for dictionary learning eliminates the need for data truncation and avoids issues such as atom splitting.  

This algorithm library provides two fast dictionary learning algorithms: (1) Fast_CSDL and (2) CL_CSDL

# FastCSDL
a fast convolutional sparse dictionary learning algorithm, It can be applied to long one-dimensional signal processing. It doesn't need to divide long data into small segments to speed up the algorithm. It has low memory consumption and high computational efficiency.  

# CLCSDL
An improved version of fast convolutional sparse dictionary learning, further reducing memory consumption and improving computational speed.
